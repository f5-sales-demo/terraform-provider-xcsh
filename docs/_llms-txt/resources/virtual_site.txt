# xcsh_virtual_site

Category: Sites
Virtual site object in given namespace

## Required

- name
- namespace
- site_type
- site_selector

## Minimal Valid Config

```terraform
resource "xcsh_virtual_site" "example" {
  name      = "example-virtual-site"
  namespace = "staging"

  site_type = "INVALID"
}
```

## Dependencies

Requires: none
