# xcsh_dns_lb_health_check

Category: DNS
DNS Load Balancer Health Check in a given namespace. If one already exist it will give a error

## Required

- name

## Minimal Valid Config

```terraform
resource "xcsh_dns_lb_health_check" "example" {
  name      = "example-dns-lb-health-check"
  namespace = "system"
}
```

## Dependencies

Requires: none
