# xcsh_azure_vnet_site

Category: Sites
Deploying F5 sites within Azure Virtual Network environments

## Required

- name
- namespace
- machine_type
- resource_group
- ssh_key

## Minimal Valid Config

```terraform
resource "xcsh_azure_vnet_site" "example" {
  name      = "example-Azure-vnet-site"
  namespace = "staging"

  machine_type   = "example-value"
  resource_group = "example-value"
  ssh_key        = "example-value"
}
```

## Dependencies

Requires: none
