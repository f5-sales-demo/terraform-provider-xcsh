# xcsh_fast_acl_rule

Category: Security
New Fast ACL rule, has specification to match source IP, source port and action to apply

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_fast_acl_rule" "example" {
  name      = "example-fast-acl-rule"
  namespace = "staging"
}
```

## Dependencies

Requires: none
