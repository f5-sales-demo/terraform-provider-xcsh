# xcsh_irule

Category: BIG-IP Integration
IRule in a given namespace. If one already exists it will give an error

## Required

- name
- namespace
- description
- description_spec
- irule

## Minimal Valid Config

```terraform
resource "xcsh_irule" "example" {
  name      = "example-irule"
  namespace = "staging"

  description_spec = "example-value"
  irule            = "example-value"
  description      = "example-value"
}
```

## Dependencies

Requires: none
