# f5xc_proxy

Category: Networking
Tcp loadbalancer create specification. configuration

## Required

- name
- namespace
- connection_timeout

## Minimal Valid Config

resource "f5xc_proxy" "example" {
  name      = "example-proxy"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  proxy_url = "`http://proxy.example.com:8080"`
}

## Dependencies

Requires: none
