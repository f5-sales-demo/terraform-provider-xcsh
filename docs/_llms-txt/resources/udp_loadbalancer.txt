# xcsh_udp_loadbalancer

Category: Load Balancing
Load balancing UDP traffic across origin pools

## Required

- name
- namespace
- dns_volterra_managed
- enable_per_packet_load_balancing
- idle_timeout

## Minimal Valid Config

```terraform
resource "xcsh_udp_loadbalancer" "example" {
  name      = "example-udp-loadbalancer"
  namespace = "staging"

  domains                          = ["example-value"]
  dns_volterra_managed             = true
  enable_per_packet_load_balancing = true
  idle_timeout                     = 1
}
```

## Dependencies

Requires: namespace, origin_pool
