# xcsh_tunnel

Category: Networking
Tunnel in a given namespace. If one already exist it will give a error

## Required

- name
- namespace
- tunnel_type

## Minimal Valid Config

```terraform
resource "xcsh_tunnel" "example" {
  name      = "example-tunnel"
  namespace = "staging"

  tunnel_type = "IPSEC_PSK"
}
```

## Dependencies

Requires: none
