# xcsh_cloud_elastic_ip

Category: Cloud Resources

- item_count

## Minimal Valid Config

```terraform
resource "xcsh_cloud_elastic_ip" "example" {
  name      = "example-cloud-elastic-ip"
  namespace = "system"

  item_count = 1
}
```
