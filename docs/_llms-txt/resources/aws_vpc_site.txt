# xcsh_aws_vpc_site

Category: Sites
Deploying F5 sites within AWS VPC environments

## Required

- name
- address
- aws_region
- disk_size
- instance_type
- ssh_key

## Minimal Valid Config

```terraform
resource "xcsh_aws_vpc_site" "example" {
  name      = "example-aws-vpc-site"
  namespace = "system"

  aws_region    = "example-value"
  instance_type = "example-value"
  ssh_key       = "example-value"
  address       = "example-value"
  disk_size     = 1
}
```

## Dependencies

Requires: none
