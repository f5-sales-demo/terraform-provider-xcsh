# xcsh_service_policy_rule

Category: Security
Service_policy_rule creates a new object in the storage backend for metadata.namespace

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_service_policy_rule" "example" {
  name      = "example-service-policy-rule"
  namespace = "staging"
}
```
