# xcsh_sensitive_data_policy

Category: Security
Sensitive_data_policy creates a new object in the storage backend for metadata.namespace

- namespace

## Minimal Valid Config

```terraform
resource "xcsh_sensitive_data_policy" "example" {
  name      = "example-sensitive-data-policy"
  namespace = "staging"
}
```
