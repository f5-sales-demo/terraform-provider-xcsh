# f5xc_virtual_k8s

Category: Kubernetes
Virtual_k8s will create the object in the storage backend for namespace metadata.namespace

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "f5xc_virtual_k8s" "example" {
  name      = "example"
  namespace = "system"

  labels = {
    "env" = "production"
  }
}
```

## Dependencies

Requires: none
