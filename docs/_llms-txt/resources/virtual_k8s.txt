# xcsh_virtual_k8s

Category: Kubernetes
Virtual_k8s will create the object in the storage backend for namespace metadata.namespace

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_virtual_k8s" "example" {
  name      = "example-virtual-k8s"
  namespace = "staging"
}
```

## Dependencies

Requires: none
