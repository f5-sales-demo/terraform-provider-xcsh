# xcsh_healthcheck

Category: Load Balancing
Healthcheck object defines method to determine if the given endpoint is healthy. single healthcheck object can be referred to by one or many cluster objects. configuration

## Required

- name
- namespace
- healthy_threshold
- interval
- timeout
- unhealthy_threshold

## Server Defaults (safe to omit)

- jitter_percent

## Minimal Valid Config

```terraform
resource "xcsh_healthcheck" "example" {
  name      = "example-healthcheck"
  namespace = "staging"

  healthy_threshold   = 1
  interval            = 1
  timeout             = 1
  unhealthy_threshold = 1
}
```

## Dependencies

Requires: namespace
Used by: origin_pool
