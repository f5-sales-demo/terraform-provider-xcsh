# xcsh_fleet

Category: Sites
Fleet will create a fleet object in 'system' namespace of the user

## Required

- name
- namespace
- enable_default_fleet_config_download
- fleet_label
- operating_system_version
- volterra_software_version

## Minimal Valid Config

```terraform
resource "xcsh_fleet" "example" {
  name      = "example-fleet"
  namespace = "staging"

  fleet_label                          = "example-value"
  enable_default_fleet_config_download = true
  operating_system_version             = "example-value"
  volterra_software_version            = "example-value"
}
```

## Dependencies

Requires: none
