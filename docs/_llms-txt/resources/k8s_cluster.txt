# xcsh_k8s_cluster

Category: Kubernetes
K8s_cluster will create the object in the storage backend for namespace metadata.namespace

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_k8s_cluster" "example" {
  name      = "example-k8s-cluster"
  namespace = "staging"
}
```

## Dependencies

Requires: none
