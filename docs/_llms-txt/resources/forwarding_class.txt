# xcsh_forwarding_class

Category: Networking
Forwarding class is created by users in system namespace. configuration

## Required

- name
- namespace
- queue_id_to_use
- interface_group

## Minimal Valid Config

```terraform
resource "xcsh_forwarding_class" "example" {
  name      = "example-forwarding-class"
  namespace = "staging"

  interface_group = "ANY_AVAILABLE_INTERFACE"
  queue_id_to_use = "DSCP_BEST_EFFORT"
}
```

## Dependencies

Requires: none
