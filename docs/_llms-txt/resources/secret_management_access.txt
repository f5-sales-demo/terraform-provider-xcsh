# xcsh_secret_management_access

Category: Authentication
Secret_management_access creates a new object in storage backend for metadata.namespace

## Required

- name
- provider_name

## Minimal Valid Config

```terraform
resource "xcsh_secret_management_access" "example" {
  name      = "example-secret-management-access"
  namespace = "system"

  provider_name = "example-value"
}
```

## Dependencies

Requires: none
