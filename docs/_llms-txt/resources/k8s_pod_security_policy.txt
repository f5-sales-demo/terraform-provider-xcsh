# f5xc_k8s_pod_security_policy

Category: Security
K8s_pod_security_policy will create the object in the storage backend for namespace metadata.namespace

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "f5xc_k8s_pod_security_policy" "example" {
  name      = "example"
  namespace = "system"

  labels = {
    "env" = "production"
  }
}
```

## Dependencies

Requires: none
