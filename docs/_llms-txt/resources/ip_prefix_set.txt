# xcsh_ip_prefix_set

Category: Networking
Ip_prefix_set creates a new object in the storage backend for metadata.namespace

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_ip_prefix_set" "example" {
  name      = "example-ip-prefix-set"
  namespace = "staging"
}
```

## Dependencies

Requires: none
