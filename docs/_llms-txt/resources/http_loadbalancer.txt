# xcsh_http_loadbalancer

Category: Load Balancing
Load balancing HTTP/HTTPS traffic with advanced routing and security

## Required

- name
- namespace
- domains

## Server Defaults (safe to omit)

- connection_timeout
- http_idle_timeout

## Minimal Valid Config

```terraform
resource "xcsh_http_loadbalancer" "example" {
  name      = "example-http-loadbalancer"
  namespace = "staging"

  domains = ["example-value"]
}
```

## Dependencies

Requires: namespace, origin_pool
Used by: route
