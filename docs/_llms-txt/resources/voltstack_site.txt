# xcsh_voltstack_site

Category: Sites
Deploying Volterra stack sites for edge computing

## Required

- name
- address
- volterra_certified_hw

## Minimal Valid Config

```terraform
resource "xcsh_voltstack_site" "example" {
  name      = "example-voltstack-site"
  namespace = "system"

  volterra_certified_hw = "example-value"
  worker_nodes          = ["example-value"]
  address               = "example-value"
}
```

## Dependencies

Requires: none
