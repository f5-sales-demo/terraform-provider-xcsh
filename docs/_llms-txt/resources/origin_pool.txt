# xcsh_origin_pool

Category: Load Balancing
Defining backend server pools for load balancer targets

## Required

- name
- namespace
- origin_servers
- port

## Server Defaults (safe to omit)

- connection_timeout
- http_idle_timeout

## Minimal Valid Config

```terraform
resource "xcsh_origin_pool" "example" {
  name      = "example-origin-pool"
  namespace = "staging"
}
```

## Dependencies

Requires: namespace, healthcheck
Used by: http_loadbalancer, tcp_loadbalancer, udp_loadbalancer
