# xcsh_gcp_vpc_site

Category: Sites
Deploying F5 sites within Google Cloud VPC environments

## Required

- name
- address
- disk_size
- gcp_region
- instance_type
- ssh_key

## Minimal Valid Config

```terraform
resource "xcsh_gcp_vpc_site" "example" {
  name      = "example-gcp-vpc-site"
  namespace = "system"

  gcp_region    = "example-value"
  instance_type = "example-value"
  ssh_key       = "example-value"
  address       = "example-value"
  disk_size     = 1
}
```

## Dependencies

Requires: none
