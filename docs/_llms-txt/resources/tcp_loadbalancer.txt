# xcsh_tcp_loadbalancer

Category: Load Balancing
Load balancing TCP traffic across origin pools

## Required

- name
- namespace
- origin_pools

## Minimal Valid Config

```terraform
resource "xcsh_tcp_loadbalancer" "example" {
  name      = "example-tcp-loadbalancer"
  namespace = "staging"
}
```

## Dependencies

Requires: namespace, origin_pool
